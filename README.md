## Development

### Installation

1. Install TK 

Mac instructions:
```
brew install tcl-tk
```

2. Clone the repo & install python dependencies:
```sh
git clone git@github.com:CharlesTaylor7/turtles.git
cd turtles
pip3 install -r requirements.txt
```

### Run

```sh
python3 run.py news
```
